Coloque o projeto dentro da pasta ‘htdocs’ do seu servidor Apache. Abra-o através de qualquer browser em localhost.

Documentação:
	getArtist();

Parâmetros: String.
Retorno: Objeto JSON.
	getArtistName();

Parâmetros: Objeto.
Retorno: String.
